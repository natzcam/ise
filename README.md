# ise

## Development

### Server
1. jdk1.8.0_144
2. Apache Maven 3.5.0

```sh
cd server
./build #build classes and jars
./test  #run tests
./run   #run fhir server @ http://localhost:8888/fhir
```

### Frontend
1. Node v8.12.0

```sh
cd frontend
npm install #install dependencies
npm start   #run frontend dev server @ http://localhost:4200
```
